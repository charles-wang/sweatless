Classpublic class SmartMovieClip
InheritanceSmartMovieClip SmartSprite flash.display.Sprite

The SmartMovieClip is a substitute class for the native MovieClip class, but need sets the source. It extends SmartSprite also the all properties of the native MovieClip class and more. Remember the SmartMovieClip don't have scenes.
