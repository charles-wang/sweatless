override public function clone():Event

Returns a new CustomEvent object that is a copy of the original instance of the CustomEvent object. You do not normally call clone(); the EventDispatcher class calls it automatically when you redispatch an event that is, when you call dispatchEvent(customevent) from a handler that is handling event. The new CustomEvent object includes all the properties of the original.

Returns
Event — A new CustomEvent object that is identical to the original.
