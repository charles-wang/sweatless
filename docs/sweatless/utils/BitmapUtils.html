public static function convertToBitmap(p_target:DisplayObject, p_margin:int = 5, p_smoothing:Boolean = true, p_remove:Boolean = true, p_crop:Rectangle = null):Bitmap

Draws the target DisplayObject into a Bitmap.

Parameters

p_target:DisplayObject — The DisplayObject to be drawn.
 
p_margin:int (default = 5) — The margin to be added to the DisplayObject width and height.
 
p_smoothing:Boolean (default = true) — Defines if the resulting Bitmap should be smoothed or not.
 
p_remove:Boolean (default = true) — Defines if the target DisplayObject should be removed from stage or not.
 
p_crop:Rectangle (default = null) — The area to crop.

Returns
Bitmap — The resulting Bitmap object.
