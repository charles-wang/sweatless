render()method 
public function render(p_random:int = -1, p_min:uint = 0, p_max:uint = 0xFFFFFF, p_channel:uint, p_grayscale:Boolean = true):void

Running noise.

Parameters

p_random:int (default = -1) — The random seed number to use. If you keep all other parameters the same, you can generate different pseudo-random results by varying the random seed value. The noise function is a mapping function, not a true random-number generation function, so it creates the same results each time from the same random seed.
 
p_min:uint (default = 0) — The lowest value to generate for each channel (0 to 255).
 
p_max:uint (default = 0xFFFFFF) — The highest value to generate for each channel (0 to 255).
 
p_channel:uint (default = NaN) — A number that can be a combination of any of the four color channel values (BitmapDataChannel.RED, BitmapDataChannel.BLUE, BitmapDataChannel.GREEN, and BitmapDataChannel.ALPHA). You can use the logical OR operator (|) to combine channel values.
 
p_grayscale:Boolean (default = true) — A Boolean value. If the value is true, a grayscale image is created by setting all of the color channels to the same value. The alpha channel selection is not affected by setting this parameter to true.
